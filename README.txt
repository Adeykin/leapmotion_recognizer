Russian sign language recognizer based on Leap motion controller data.

=== Installation ===
1) Python 2.7, wxPython
2) Install Leap motion drivers. 
    Linux: Leap-2.1.4+22333-x86.deb
3) Download Leap motion SDK. Copy files sdk/Leap.py, sdk/lib/LeapPython, sdk/lib/libLeap.so into project folder

=== Run ===
$ python2.7 staticGestureWriter.py
